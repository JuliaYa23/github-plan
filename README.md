##Работа с Github


*GitHub* — сама популярна платформа для хранения IT-проектов и командной работы над ними.

Чтобы Git начал отслеживать изменения в проекте, папку с файлами этого проекта нужно сделать Git-репозиторием.

**Шаги**:



1. Для этого нужно создать папку в консоли в домашней директории, сделать ее репозиторием, создать в ней папку. В папку внести информацию, далее добавить ее к сохранению, создать коммит (сохранить ее), запушить.

2. Открыть GitHub, проверить как локальный репозиторий синхронизировался с удаленным.
